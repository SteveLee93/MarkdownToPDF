## 권장 학습 방법

### 빠른 파악 (30분)
1단계 → 2단계 → 3단계만 읽으면 전체 그림이 잡힙니다.

### 표준 학습 (1.5시간)
1단계부터 7단계까지 순서대로 읽습니다.

### 실무 투입 전 (2시간+)
모든 문서를 읽은 후, 4단계(초기화 흐름)의 코드 경로를 직접 따라가며 소스를 확인합니다.

## 프로젝트 한줄 요약

**rodi-x-svc**는 로봇 제어 플랫폼(RODI)에서 **플러그인(ASAR 아카이브) 기반 확장 기능**을 관리하는 Node.js 서비스입니다. 플러그인을 로드하고, MQTT 이벤트를 통해 다른 서비스와 통신하며, 프로그램 노드/확장/위젯/데몬의 라이프사이클을 관리합니다.
